Privacy Policy for StoryTok

Effective Date: [Insert Date]

1. Introduction

StoryTok is an automation tool that generates and posts videos on TikTok using AI-generated content.

2. Data Collection

StoryTok does not collect, store, or process any personal user data. The only interaction with user accounts is through the TikTok API for video uploads.

3. Data Usage

StoryTok uses TikTok's API solely for automating the posting of videos. No user data is stored, shared, or sold.

4. Third-Party Services

StoryTok uses the following third-party services:

  • OpenAI API (for AI-generated stories)
  • ElevenLabs API (for AI-generated voiceovers)
  • TikTok API (for video uploads)

These services may have their own privacy policies.

5. User Rights

Since StoryTok does not collect personal data, users do not need to take any action regarding their privacy.
